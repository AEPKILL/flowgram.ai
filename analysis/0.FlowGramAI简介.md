# FlowGramAI 简介

AI 浪潮席卷而来，工作流工具成为了其中的重要一环。

目前构建 AI 工作流主要有两种方式:

- **可视化搭建**: 如 Dify、Coze 等，通过拖拽节点的方式构建流程
- **代码编排**: 如 [Langchain](https://www.langchain.com/)、[AI SDK](https://ai-sdk.dev/) 等，通过编写代码来定义流程

代码方式虽然灵活强大，但对非程序员不够友好，调试也不够直观。因此，大多数企业会选择可视化方案来构建业务工作流。

但从零开始构建一个可视化工作流应用并不容易，核心挑战在于:

**画布交互**: 如何优雅地编排节点和连线?
布局算法、框选、拖拽、撤销重做、分组……这些交互细节都需要精心设计。

**数据管理**: 如何处理节点间的数据流转?
可以把工作流理解为可视化的代码——每个节点是一个函数，有自己的作用域和变量，还可能继承父级上下文。实际开发中，数据校验、联动、副作用处理、错误捕获等能力都不可或缺。

FlowGramAI 正是为了解决这些问题而生。它是一个工作流开发框架与工具集，帮助开发者快速构建生产级的工作流应用。
