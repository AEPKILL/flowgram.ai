# 快捷键

## 自定义快捷键

```ts pure
// 添加到 EditorProps
{
  shortcuts(shorcutsRegistry: ShortcutsRegistry, ctx) {
      // 按住 cmmand + a，选中所有节点
      shorcutsRegistry.addHandlers({
        commandId: 'selectAll',
        shortcuts: ['meta a', 'ctrl a'],
        execute() {
          const allNodes = ctx.document.getAllNodes();
          ctx.playground.selectionService.selection = allNodes;
        },
      });
  },
}

```
